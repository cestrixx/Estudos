Ordens_de_servico - Cadastrar uma ordem de serviço

2.0.0 
Requisição para cadastro de ordem de serviço

POST
https://api.vhsys.com/v2/ordens-servico
cURL
PHP
curl -X POST \
    https://{api_address}/v2/ordens-servico \
    -H 'access-token: SEU (Access-Token) DE ACESSO ' \
    -H 'secret-access-token: SEU (Secret-Access-Token) DE ACESSO' \
    -H 'cache-control: no-cache' \
    -H 'content-type: application/json' \
    -d '{
        "id_cliente" : "123",
        "nome_cliente" : "Nome do cliente",
        "vendedor_pedido" : "Nome do vendedor",
        "vendedor_pedido_id" : "123",
        "valor_total_despesas" : "00.00",
        "valor_total_desconto" : "00.00",
        "data_pedido" : "0000-00-00",
        "data_entrega" : "0000-00-00",
        "data_realizacao" : "0000-00-00",
        "data_realizacao_hora" : "00:00",
        "garantia_ordem" : "15 Dias",
        "equipamento_ordem" : "Equipamentos usados",
        "problema_ordem" : "Problemas encontrados",
        "recebimento_ordem" : "Detalhes sobre o recebimento",
        "referencia_ordem" : "Referência",
        "obs_pedido" : "Observação",
        "obs_interno_pedido" : "Observação interna",
        "laudo_ordem" : "Laudo sobre o serviço ou produto",

    }'

2.0.0 
Requisição para a consulta de diversas ordens de serviço

GET
https://api.vhsys.com/v2/ordens-servico
cURL
PHP
curl -X GET \
    https://{api_address}/v2/ordens-servico \
    -H 'access-token: SEU (Access-Token) DE ACESSO ' \
    -H 'secret-access-token: SEU (Secret-Access-Token) DE ACESSO' \
    -H 'cache-control: no-cache' \
    -H 'content-type: application/json' \
Parâmetro

Campo	Tipo	Descrição
order opcional	String	
Nome do campo para ordenação EX:data_mod_pedido

sort opcional	String	
Tipo de ordenação

Valor padrão: Asc

Valores permitidos: Asc, Desc

limit opcional	Integer	
Limit de registros

Valor padrão: 250

Faixa de tamanho: 0-250

offset opcional	Integer	
Registro inicial da consulta

nome_cliente opcional	String	
Nome do cliente

vendedor opcional	String	
Nome do vendedor

valor_produtos opcional	String	
Valor total dos produtos
